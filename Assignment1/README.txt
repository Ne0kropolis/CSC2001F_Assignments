Name : Mutual Friend Searcher(Assignment 1)
Description: Mutual Friend Searcher is a java class used to search for names in one text file, in another file using a sequential method and a binary search method. It lists the positions(if any) of the names in the files and provides the time taken to compare the two search methods.
Instructions: Compile and run Searcher.java, with toSearchFor.txt and toSearchIn.txt in the same folder. The methods can also be called seperately, and the text files can be changed in lines 111 and 124.
List of Files: Searcher.java, toSearchFor.txt, toSearchIn.txt
